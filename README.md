# RADESMEE ENCODER

## how to install

```bash
cargo install radesmee_encoder
```

[see crates.io package](https://crates.io/crates/radesmee_encoder)

## how to use

to get a list of commands and their description:

```bash
radesmee_encoder --help
```

or

```bash
radesmee_encoder -h
```

to encode utf8 text into binary radesmee/radaesmee code:

```bash
radesmee_encoder crypt "your text"
```

to decode binary radesmee/radaesmee code into utf8 text

```bash
radesmee_encoder decrypt "your radaesmee code"
```
